# bank client segmentation credit limit regression

Анализ данных клиентов банка для выделения портретов клиентов и построения прогностической модели для определения суммы кредитного лимита.

## 📋 Описание проекта
Проведен полный цикл анализа данных клиентов банка (170 746 записей, 18 признаков) для выявления ключевых характеристик и различий между группами клиентов, открывающих и не открывающих кредитную карту.

**Цель:** Сегментировать клиентов, выявить ключевые факторы, влияющие на открытие карты, и разработать рекомендации для маркетингового отдела.

## 🛠 Стек технологий
- **Python:** Pandas (обработка данных), NumPy (математические операции), Matplotlib/Seaborn (визуализация)
- **Jupyter Notebook:** Интерактивная среда разработки
- **Git/GitHub:** Контроль версий и хостинг проекта

## 🚀 Ключевые этапы работы

### 1️⃣ Предобработка данных (Pandas)
- Нормализация названий столбцов, исправление форматов данных
- Заполнение пропусков: медиана для числовых признаков, нули для кредитных метрик, мода для категорий
- Выявление выбросов в сумме кредита с помощью Boxplot

### 2️⃣ Исследовательский анализ и сегментация (EDA)
- Разделение признаков на числовые и категориальные
- Создание возрастных групп: 18-24, 25-34, 35-44, 45-54, 55+
- Создание групп по доходу: Низкий, Средний, Высокий
- Сравнительный анализ групп клиентов (открывших и не открывших карту)

### 3️⃣ Визуализация и интерпретация (Matplotlib/Seaborn)
- Гистограммы распределения числовых признаков
- Столбчатые диаграммы для категориальных признаков в разрезе двух групп

### 💡 Ключевые инсайты
- **Сильные предикторы:** Наиболее значимыми факторами оказались **скоринг**, **образование** и **пол**
- **Продуктовые предпочтения:** Открывающие карту чаще пользуются тарифом `id=2`
- **Профессиональные группы:** Клиенты с кодом профессии `10` значительно чаще открывают карты

### 📌 Рекомендации для бизнеса
- **Для сегмента открывающих карту:** Предлагать премиальные карты с бонусами, targeting на женщин с высшим образованием и хорошей кредитной историей
- **Для сегмента не открывающих карту:** Разработать специальные предложения для привлечения мужской аудитории и клиентов со средним образованием

### 📈 Регрессионный анализ и прогнозирование

* **Моделирование:** Построена модель **Random Forest Regressor** для предсказания суммы кредита (`credit_sum`) на основе профиля клиента.
* **Feature Engineering:** В модель включены созданные группы по возрасту и доходу, что позволило повысить точность прогноза.
* **Ключевые факторы:** Подтверждено, что наибольший вклад в определение суммы вносят **ежемесячный доход**, **срок кредита** и **тарифный план**.
* **Оценка качества:** Модель показывает высокую надежность на массовом сегменте (кредиты до 75 000 руб.), что позволяет автоматизировать процесс назначения кредитных лимитов.
