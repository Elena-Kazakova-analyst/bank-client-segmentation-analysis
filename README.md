💳 Анализ данных банковских клиентов
Портретный анализ и сегментация клиентов для кредитных карт

📋 Описание проекта
Проведен полный цикл исследовательского анализа данных клиентов банка (170 746 записей, 18 признаков).

Цель: Выявить ключевые различия между клиентами, открывающими и не открывающими кредитную карту, сформировать их детальные портреты и предложить рекомендации для маркетингового отдела.

🛠 Стек технологий
Python: Pandas (предобработка), NumPy (математические операции), Matplotlib/Seaborn (визуализация распределений).

Обработка данных: Методы заполнения пропусков (медиана, мода), корректировка типов, работа с выбросами.

Аналитика: Сравнительный анализ групп (группировка по open_account_flg), сегментация по возрасту и доходу, кросс-анализ сегментов.

🚀 Ключевые этапы работы
1️⃣ Предобработка и очистка данных (Pandas)

Нормализация названий столбцов, исправление форматов данных (замена запятых на точки в числах, приведение строк к датам).

Анализ и заполнение пропусков: для числовых признаков использована медиана (устойчивая к выбросам), для кредитных метрик (credit_count) – нули, для категорий – мода.

Выявление выбросов в сумме кредита с помощью Boxplot (межквартильный размах), что показало наличие клиентов с аномально высокими кредитами.

2️⃣ Исследовательский анализ и сегментация (EDA)

Разделение признаков: Числовые и категориальные переменные проанализированы отдельно для выявления закономерностей.

Сегментация базы: Созданы новые признаки для более глубокого анализа:

Возрастные группы: 18-24, 25-34, 35-44, 45-54, 55+.

Группы по доходу: Низкий, Средний, Высокий.

Сравнение групп: Проведен детальный сравнительный анализ средних, медиан и частот по каждой группе клиентов (открывших и не открывших карту).

3️⃣ Визуализация и интерпретация (Matplotlib/Seaborn)

Построены гистограммы распределения всех числовых признаков для оценки формы данных.

Созданы столбчатые диаграммы (countplot), наглядно демонстрирующие различия в категориальных признаках (пол, образование, семейный статус) между двумя группами клиентов.
